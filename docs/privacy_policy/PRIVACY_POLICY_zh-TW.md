# Custom Script Injector 隱私權政策

**最後更新日期：** 2025 年 12 月 10 日  
**作者：** GeminiCLI Gemini 2.0 Flash Thinking Experimental

## 簡介

Custom Script Injector（以下簡稱「我們」、「本擴充功能」）致力於保護您的隱私。本隱私權政策說明我們在您使用本 Chrome 擴充功能時如何處理您的資訊。

## 我們收集的資訊

### 本地儲存的資訊

Custom Script Injector 僅在**您的裝置本地**儲存以下資訊：

- **自訂腳本**：您為特定網域建立的 CSS 和 JavaScript 程式碼
- **網域別名**：您為網域指定的自訂名稱
- **GitHub 個人存取權杖**：如果您選擇使用 GitHub Gist 同步功能（儲存在本地儲存空間，絕不同步）
- **語言偏好設定**：您選擇的介面語言

### 我們不收集的資訊

我們想明確說明我們**不會**做的事情：

- ❌ 我們**不會**收集任何個人資訊
- ❌ 我們**不會**追蹤您的瀏覽記錄
- ❌ 我們**不會**將任何資料傳送到外部伺服器（除非您明確啟用 GitHub Gist 同步）
- ❌ 我們**不會**使用分析或追蹤工具
- ❌ 我們**不會**與第三方分享您的資料

## 我們如何使用您的資訊

所有資料都儲存在您的裝置本地，僅用於：

1. **執行自訂腳本**：將您的 CSS 和 JavaScript 套用到您指定的網站
2. **跨裝置同步**（選用）：如果您啟用 GitHub Gist 同步，您的腳本將上傳到您的私人 GitHub Gist 帳戶

## 第三方服務

### GitHub Gist（選用）

如果您選擇使用備份/還原功能：

- 您必須提供 GitHub 個人存取權杖
- 您的腳本將上傳到您 GitHub 帳戶中的**私人 Gist**
- 我們無法存取您的 GitHub 帳戶或權杖
- 請參閱 [GitHub 隱私權政策](https://docs.github.com/zh/site-policy/privacy-policies/github-privacy-statement)

## 資料安全

- 所有資料都使用 Chrome 的 `chrome.storage.local` API 儲存在本地
- GitHub 權杖儲存在本地儲存空間，絕不透過 Chrome 同步功能同步
- 我們使用安全的 API（`declarativeNetRequest`、`scripting`）來注入腳本

## 您的權利

您對自己的資料擁有完全控制權：

- **存取**：在選項頁面中檢視所有腳本
- **修改**：隨時編輯或刪除任何腳本
- **匯出**：將腳本備份到 GitHub Gist
- **刪除**：解除安裝擴充功能會移除所有本地儲存的資料

## 兒童隱私

本擴充功能不適用於 13 歲以下的兒童。我們不會故意收集兒童的資訊。

## 隱私權政策變更

我們可能會不時更新本隱私權政策。任何變更都會反映在上方的「最後更新日期」中。

- 透過電子郵件聯絡我們（如果 Chrome 線上應用程式商店清單中有提供）
